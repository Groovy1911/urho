1. Install MinGW + addons : gcc & msys
	1. Add to the path
		C:\MinGW\cmake-3.7.1-win32-x86\bin
		C:\MinGW\bin
		ANDROID_NDK = C:\MinGW\android-ndk-r12b
2. Create folders into "urho\Bin\Android":
	arm64-v8a
	armeabi
	armeabi-v7a
	x86
	x86_x64
3. Replace "urho\Urho3D\Source\Source\Urho3D\CMakeLists.txt" by "urho\Tools\CMakeLists.txt"
3. Start "msys.bat"
4. Start command "make -j5 Android"
