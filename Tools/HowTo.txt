1. Install MinGW + addons : gcc & msys & cmake (https://cmake.org/download/ --> Binary distributions: Windows win32-x86 ZIP)
	1. Add to the path
		C:\MinGW\cmake-x.x.x-win32-x86\bin
		C:\MinGW\bin
		ANDROID_NDK = C:\MinGW\android-ndk-r12b
2. Create folders into "urho\Bin\Android":
	arm64-v8a
	armeabi
	armeabi-v7a
	x86
	x86_x64
3. Replace "urho\Urho3D\Source\Source\Urho3D\CMakeLists.txt" by "urho\Tools\CMakeLists.txt"
4. Copy "urho\Tools\Android.cmake" to "urho\Urho3D\Source\CMake\Toolchains"
5. Start "msys.bat"
6. Start command "make -j5 Android"
